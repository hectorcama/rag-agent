# RAG Agent

This is a personal project to understand how RAG (Retrieval-Augmented Generation) works and then potentially provide a RAG agent to my company that uses scientific knowledge.

## Installation

This project uses a `pyproject.toml` and poetry configuration.

To install the project and its dependencies:

1.  **Install Poetry**: Follow the instructions at [python-poetry.org](https://python-poetry.org/docs/#installation).
2.  **Install dependencies**:
    ```bash
    poetry install
    ```
3.  **Set up pre-commit hooks**:
    ```bash
    poetry run pre-commit install
    ```
